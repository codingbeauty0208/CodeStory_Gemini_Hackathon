### Slide 2: Key Libraries and Tools
-   **Web Server Gateway Interfaces**: Employs `asgiref` for ASGI (Asynchronous Server Gateway Interface) support and `wsgi.py` for WSGI (Web Server Gateway Interface), enabling compatibility with various web servers.
-   **Image Processing**: Integrates **Pillow**, a fork of the Python Imaging Library (PIL), for image manipulation and processing, likely used for handling uploaded images for hotels or user profiles.
-   **SQL Utilities**: Includes `sqlparse` for parsing SQL statements, typically used for debugging or formatting SQL queries.
-   **Timezone Management**: Leverages `pytz` for accurate timezone calculations, essential for handling dates and times across different geographical locations, relevant for travel applications.
