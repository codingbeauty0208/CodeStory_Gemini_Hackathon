### Slide 1: Core Frameworks and Languages
-   **Backend Framework**: Predominantly built using **Django**, a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Evidenced by `manage.py`, `settings.py`, and `requirements.txt`.
-   **Programming Language**: The entire backend logic and application structure are implemented using **Python 3**.
-   **Database**: Utilizes **SQLite 3** as its default database backend, as configured in `settings.py` (`'ENGINE': 'django.db.backends.sqlite3'`). This choice suggests ease of setup and local development.
