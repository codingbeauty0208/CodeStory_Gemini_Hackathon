### Slide 1: Backend Technologies
- **Ruby Language**: The application is built using Ruby, specifically version `3.1.3`, as defined in `.ruby-version` and `Gemfile`.
- **Ruby on Rails Framework**: The core web framework is Ruby on Rails, version `~> 7.0.4, >= 7.0.4.3`, as specified in `Gemfile`.
- **Web Server**: `Puma` (version `~> 5.0`) is employed as the default web server, as seen in `Gemfile` and `config/puma.rb`.
- **Database**: `MySQL` is the chosen database, indicated by the `mysql2` gem (version `~> 0.5`) in `Gemfile` and configurations in `config/database.yml`.
