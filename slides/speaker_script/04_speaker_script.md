### Slide 4
Moving on to the technology stack that powers this application. The core language used throughout the project is Java, a robust and widely adopted language for enterprise solutions. The primary framework is Spring Boot, which significantly simplifies the development of production-ready Spring applications. For persistent data management, the project leverages Spring Data JPA, providing an elegant way to interact with databases. The chosen database is H2, specifically configured as an in-memory solution, making it ideal for development, testing, and rapid prototyping without the need for an external database setup. Lastly, Apache Maven is the build tool, managing project dependencies and the entire build lifecycle.
